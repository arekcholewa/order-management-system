# Order Management System

A simple REST API for managing products and customer orders, built with Spring Boot.

## Overview

The application allows users to:

- Create and retrieve products
- Create customer orders
- Store order items associated with products
- Manage customer and order data

The project uses an in-memory H2 database and Liquibase for database schema management.

## Technologies

- Java 17+
- Spring Boot
- Spring Data JPA
- Hibernate
- Liquibase
- H2 Database
- Maven
